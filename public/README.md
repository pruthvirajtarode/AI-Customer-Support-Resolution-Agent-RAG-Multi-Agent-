# SupportAI SaaS - HTML5 Frontend

This is a modern, attractive HTML5/CSS3/JavaScript frontend for the SupportAI SaaS project.

## Features
- Responsive, single-page navigation
- Login, Signup, Dashboard, Policy Upload, Ticket Submission, AI Responses
- Clean, professional UI with modern CSS
- Connects to FastAPI backend (http://localhost:8000)

## Usage
1. Open `index.html` in your browser.
2. Make sure the backend is running at `http://localhost:8000`.
3. Use the navigation bar to access all features.

---

**For production, deploy these files to any static web host (Vercel, Netlify, GitHub Pages, etc).**
